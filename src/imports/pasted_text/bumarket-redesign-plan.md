Design and redesign a modern web-based student marketplace platform called “BUMarket” for Bicol University students and campus entrepreneurs. The existing system already contains a homepage, marketplace, seller dashboard, buyer dashboard, and product details page. Preserve the current visual identity, branding, color palette, and university-inspired style from the existing prototype, but evolve it into a more polished, professional e-commerce experience inspired by Shopee and Lazada, while still feeling student-friendly and campus-oriented.

The system uses a React + TypeScript + Tailwind CSS frontend with Supabase backend integration. The redesign should not feel corporate or luxury-focused. It should feel accessible, lightweight, organized, and familiar to university students.

PRIMARY OBJECTIVE:
Transform the existing prototype into a stable, scalable, user-centered marketplace platform that improves buyer-seller interaction, trust, usability, and marketplace efficiency.

DESIGN STYLE:

Keep the existing university-inspired branding and colors
Add e-commerce UX patterns similar to Shopee/Lazada
Use clean spacing, card-based layouts, rounded corners, modern shadows, and responsive structure
Prioritize usability and clarity over visual complexity
Maintain a student-friendly atmosphere
Avoid overdesigned enterprise dashboards
Use intuitive navigation and marketplace hierarchy
